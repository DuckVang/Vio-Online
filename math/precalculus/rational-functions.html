Rational functions

”ratio of polynomials”- a rational function is a function that can be constructed from a variable and a set of numbers, using addition, multiplication and division*

Simply it is a ratio of polynomialspolynomials, where the denominator shouldn’t be equal to 0

Asymptotes

An asymptote is a line being approached by a curve but never touching the curve i.e., an asymptote is a line to which the graph of a function converges

 Horizontal

This can only occur if - simply when the denominator is larger than or the same as the numerator (the highest degree of all of them) - if not then it is maybe some form of oblique/slanted asymptote

Determine location by dividing it by and assuming the is a large number (infinity) - just like finding limits

 Vertical

To find the vertical asymptote, we simplify it first to the lowest terms, set its denominator equal to zero, , and then solve for x value

Notice has no vertical asymptote at even though this is a zero of the denominator. This is because the numerator also vanishes at

Oblique

Happens when the denominator degree is lower than the numerator → need to adjusted it (separate)

Zeros of rational functions

A rational function can only intersect the x-axis if the numerator is zero

But be beware if the numerator and denominator have the same zero then this point is not in the domain of our function (see example in the vertical asymptote)

The locations of the zeros can heavily influence what the graph of a rational function looks like
